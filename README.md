# KDFGC Club Hub - Mobile Web Development Capstone

## Project Overview
A full-featured Android application and portfolio website developed for the Kelowna & District Fish and Game Club as part of the Mobile Web Development Capstone at Sundance College.

## Live Portfolio
View Portfolio Website: https://turnagain672.github.io/KDFGC_Capstone/

## Android App Features
- Member Management - Registration, login, profile management with role-based access (Admin/Moderator/Member)
- Event Scheduling - View and register for club events
- Course Registration - Browse and sign up for PAL, RPAL, and safety courses
- Range Logging - Track range visits with photo uploads
- Forum - Community discussion board
- Push Notifications - Firebase Cloud Messaging integration
- Admin Panel - Member management, invoicing, notifications, news publishing

## Tech Stack

### Android App
- Kotlin
- Jetpack Compose
- Material Design 3
- Room Database
- Retrofit for API calls
- WorkManager for background tasks
- Firebase Cloud Messaging
- BCrypt for password hashing
- JUnit and Espresso for testing

### Portfolio Website
- HTML5
- CSS3 with responsive design
- JavaScript for form validation
- GitHub Pages for deployment
